# PRD: Olaya Serena Sanchez Portfolio

## 1. Vision & Goals
A high-end, minimalist portfolio for a Swiss photographer/cinematographer. 
The goal is to showcase visual work using a "Swiss Design" aesthetic.

## 2. Design Language (The "Vibe")
- **Style:** International Typographic Style (Swiss Style).
- **Colors:** White (#FFFFFF), Marine Blue (#002045), Black (#000000).
- **Grid:** Strict but asymmetric layout.
- **Typography:** Bold Sans-Serif (Inter/Helvetica).

## 3. Core Features
- **Hero Section:** High-impact typography with the name "Olaya Serena Sanchez".
- **Dynamic Gallery:** Asymmetric grid displaying photos and video placeholders.
- **Responsive:** Must look perfect on iPhone, iPad, and Desktop.

## 4. Technical Stack
- HTML5, CSS3 (Tailwind via CDN), JavaScript (Vanilla).
- Deployment: Netlify via GitHub.
